# Packages

Shared libraries live here.

Possible future packages:

- `core`: Pantheon domain types, rules, and shared utilities.
- `agents`: agent runtime primitives.
- `assets`: asset generation and metadata workflows.
- `protocol-0g`: 0G Storage, Compute, and Chain adapters.
- `protocol-axl`: Gensyn AXL communication adapter.
- `protocol-ens`: ENS identity adapter.
- `protocol-swap`: Uniswap API adapter.
- `execution`: KeeperHub execution adapter.

Packages should stay focused and reusable across apps, examples, and tools.
